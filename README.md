# Disease Prediction from Medical Data

## 📌 Project Overview

This project predicts whether a patient is diabetic or not using Machine Learning algorithms.
The model is trained on medical data such as glucose level, BMI, blood pressure, insulin, and age.

---

## 🎯 Objective

To develop a machine learning model capable of predicting diseases using patient medical records.

---

## 🛠 Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn

---

## 📂 Dataset

Diabetes Dataset from Kaggle

Features:

* Pregnancies
* Glucose
* Blood Pressure
* Skin Thickness
* Insulin
* BMI
* Diabetes Pedigree Function
* Age

Target:

* Outcome (0 = No Diabetes, 1 = Diabetes)

---

## 🤖 Machine Learning Algorithm

Random Forest Classifier

---

## 📊 Model Accuracy

| Metric    | Score |
| --------- | ----- |
| Accuracy  | 81%   |
| Precision | 80%   |
| Recall    | 78%   |
| F1-Score  | 79%   |

---

## 📈 Confusion Matrix

The Disease Prediction model successfully predicts diabetes using patient medical data.
